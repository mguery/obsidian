# Obsidian Templates and Files

## Personal Curriculum Dashboard 
- Go to [TfTHacker](https://github.com/TfTHacker/DashboardPlusPlus/tree/master/.obsidian/snippets) and download raw files (click download button) - [dashboard.css](https://github.com/TfTHacker/DashboardPlusPlus/blob/master/.obsidian/snippets/dashboard.css) and [dashboard-ReadLineLength.css](https://github.com/TfTHacker/DashboardPlusPlus/blob/master/.obsidian/snippets/dashboard-ReadLineLength.css)
- Go to your vault - Settings > Appearance > CSS snippets > click folder icon, drag and drop .css files in there 
- Enable/toggle both files
- Go to Community Plugins, install and enable 'Dataview' plugin by Michael Brenan
- In the Dataview plugin settings page, enable those top 5 items
- Create new note called 'Home'
- Right-click the tab and pin it
- [Copy my dashboard](https://github.com/mguery/obsidian/blob/main/dashboard-pc.md) or the [original from TfTHacker](https://github.com/TfTHacker/DashboardPlusPlus/blob/master/Dashboard%2B%2B.md)
- In edit mode: Paste and edit to create your plan
- Click reading mode to view dashboard
- Use the [Course template](https://github.com/mguery/obsidian/blob/main/Course%20Name) for each course
